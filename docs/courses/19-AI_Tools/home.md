# 📝 Course Description

> This course introduces how to leverage modern AI platforms—such as OpenAI and Anthropic's Claude—to build intelligent agents and applications. Students will learn how to interface with APIs, structure prompts, and utilize features like function calling to create agents capable of executing complex workflows.

# 🎯 Prerequisites

* [Lesson 01: Introduction](/courses/01-Introduction/home.md)
* [Lesson 05: Programming](/courses/05-Programming/home.md)

# 🏆 Learning Outcomes

By the end of this course, students will be able to:

1. Understand the core concepts of AI agents and large language models.
2. Interface programmatically with the OpenAI and Anthropic APIs.
3. Build intelligent scripts that utilize AI function calling/tool usage to automate tasks.
